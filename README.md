# Proyecto Final - E-commerce React

## Descripción
Aplicación de e-commerce desarrollada con React.  
Los productos se obtienen desde Firebase Firestore y las compras generan una orden en la base de datos.

## Funcionalidades
- Listado dinámico de productos
- Vista de detalle
- Carrito con Context API
- Checkout con generación de orden en Firestore

## Tecnologías
- React
- React Router
- Firebase Firestore
- CSS
